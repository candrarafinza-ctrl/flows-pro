Sistem Monitoring Banjir

FLOWS-PRO adalah sistem berbasis IoT yang dirancang untuk memantau kondisi banjir secara real-time menggunakan sensor ketinggian air, arus, dan lingkungan. Sistem ini membantu memberikan informasi cepat dan akurat untuk mengurangi risiko bencana.

Tentang Web FLOWS-PRO

Web ini berfungsi sebagai pusat monitoring data banjir berbasis IoT yang terintegrasi dengan sensor lapangan. Data ditampilkan secara visual dan real-time untuk membantu masyarakat dan pihak terkait dalam mengambil keputusan cepat.
